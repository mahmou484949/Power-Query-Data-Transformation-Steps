# 📊 Data Cleaning & Transformation using Excel & Power Query

## 📌 Overview
This project focuses on cleaning and transforming raw data using Microsoft Excel and Power Query.  
The dataset initially contained inconsistencies, duplicated records, and formatting issues that required structured preprocessing before analysis.

---

## 🧹 Data Cleaning Process
The following steps were performed to prepare the dataset:

- Removed unnecessary columns
- Renamed columns for better readability
- Changed data types to ensure accuracy
- Handled missing and inconsistent values
- Replaced incorrect values with standardized formats
- Split columns to extract meaningful information
- Unpivoted data to convert it into a structured format
- Created conditional columns for enhanced analysis

---

- Microsoft Excel  
- Power Query
